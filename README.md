# Hi, I'm Emmanuel Gita 👋

Backend engineer focused on **payments and fintech**, and I build the full stack when a product needs it. My core work is the systems that move money: transaction lifecycles, wallets, reconciliation, and multi-gateway payment flows. I work primarily in **Laravel and PHP**, build with **Node.js / Express** when a project calls for it, and take products the rest of the way with **React and Inertia** on the frontend.

📍 Lagos, Nigeria · Open to remote

---

### What I build
- **Payment backends:** multi-gateway integrations (Stripe, Paystack, Flutterwave, Kora, Square, Cryptomus, Safehaven MFB, Reloadly), webhooks, and automated reconciliation
- **REST APIs** and third-party integrations
- **Real-time systems:** WebSocket chat, notifications, and queues (Redis)
- **Full-stack delivery:** user-facing apps and admin dashboards in React, Inertia, and Filament (e.g. Viraleety's React user app and admin)

### Live products I've shipped
- **BritEdge** (Play Store): fintech app, transfers, VTU, loans, group savings, payroll deductions
- **Bompay** (Play Store): payments app, transaction management and multi-provider VTU
- **Viraleety** ([viraleety.com](https://viraleety.com)): influencer and brand marketplace with wallet, multi-gateway payments, and automated commissions
